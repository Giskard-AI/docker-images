# docker-images

This repository builds and publishes Giskard-maintained Python Docker images.

## Python Images

Images are published to `quay.io/giskard/python`.

Supported Python versions are `3.12`, `3.13`, and `3.14`. For each version, the
publish workflow creates:

- minor tags, for example `3.13`
- patch tags, for example `3.13.9`
- platform tags, for example `3.13-ubuntu-latest` and
  `3.13-runner-ubuntu24-arm64`

Dockerfiles are generated by `python/generate_dockerfile.sh` from
`docker-library/python` fragments, then based on the pinned Ubuntu base image in
that script.

## Update Flow

The scheduled `Update python dockerfile` workflow refreshes the pinned
`ubuntu:24.04` digest and generated Python Dockerfiles, then opens or updates
the `cron-python-update` pull request.

After generated Dockerfile changes land on `main`, the `Publish the images`
workflow builds the `3.12`, `3.13`, and `3.14` images for both configured
platform runners and publishes multi-platform manifests to Quay.
